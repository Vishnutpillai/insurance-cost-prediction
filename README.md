# 🏥 Insurance Cost Prediction \| Machine Learning Project
-----------------------------------------------------------------------

## 📌 Overview

This project predicts **medical insurance charges** using machine
learning models. It analyzes how features like age, BMI, and smoking
status affect insurance cost.

------------------------------------------------------------------------

## 🎯 Objectives

-   Perform Exploratory Data Analysis (EDA)
-   Apply preprocessing techniques
-   Train multiple ML models
-   Evaluate and compare performance

------------------------------------------------------------------------

## 📂 Project Structure

insurance-project/ - insurance_analysis.ipynb - insurance.csv - README.md

------------------------------------------------------------------------

## 📊 Dataset Features

-   age
-   sex
-   bmi
-   children
-   smoker
-   region
-   charges (target)

------------------------------------------------------------------------

## ⚙️ Technologies Used

Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

------------------------------------------------------------------------

## 🤖 Models Implemented

-   Linear Regression
-   Polynomial Regression
-   Decision Tree Regressor
-   Random Forest Regressor
-   Gradient Boosting

------------------------------------------------------------------------

## 📈 Model Performance

-   Train R²: 0.74\
-   Test R²: 0.78\
-   Good generalization (no major overfitting)

------------------------------------------------------------------------

## 🔍 Key Insights

-   Smoking significantly increases insurance cost
-   BMI and age are important predictors
-   Ensemble models perform better

------------------------------------------------------------------------
